A JPA implementation of the Martin Fowler's Analysis Patterns to exemplify Object Relational Mapping.
The Book for reference:

https://martinfowler.com/books/ap.html

The following Pattern "families" are implemented:

*Account/Transaction

*Accountability/Party/Time

*Address Book

*Resource Allocation
